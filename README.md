# Meus PointFiles

Aqui estao os arquivos pontos do meu sistema (Arch by the Way).

## Requirements

Pai usa STOW, entao baixa ai.

### Stow

```
pacman -S stow
```

## Installation

Clona ai po

```
$ git clone git@github.com:r-vilela/pointfiles.git
$ cd pointfiles
```

Agora so usar o stow e para criar os SymLinks e GG.

```
$ stow.
```
